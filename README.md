<h4>Simple tool to generate flash commands for Heimdoo(mobile version of Heimdall), designed to enable Stock ROM installation on Samsung devices.</h4>

<h3>How to use:</h3>

```py
python genFlashCommand.py <PIT FILE>
  ```
